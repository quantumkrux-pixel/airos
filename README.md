AirOS
-----<br>
AirOS aims to be a true cross-platform cloud-based operating system that offers:<br>
        *  Persistent Storage<br>
        *  Instant Device-to-Device Transitioning<br>
        *  Persistent User Settings<br>
        *  Flagship OS-Level Integrated Messaging/Team Networking Application<br>
        *  Nodejs/React Compatability out-of-the-box<br>
